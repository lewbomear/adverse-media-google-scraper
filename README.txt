# Adverse Media Scraper

Adverse Media Summary is a Python script for searching, translating, and summarizing adverse media content. 

The project aims to facilitate the preliminary stages of the due diligence and third-party risk managment process by checking whether a Target has been the subject of adverse media reports and presenting the findings in an easy-to-read and review format.


## Features

- Search for relevant content using the Google Custom Search API.
- Extract information from news articles using the Newspaper library.
- Translate and summarize the content using Googletrans and Hugging Face Transformers.
- Store the results in an Excel workbook.
